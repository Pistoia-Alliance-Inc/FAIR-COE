# Pistoia Alliance
## FAIR-Community of Experts Documentation Hub

## FAIR Maturity Matrix
- An organisational maturity model of FAIR implementation.
- Link: [fairmm.pistoiaalliance.org](http://fairmm.pistoiaalliance.org/)

## FAIR Toolkit for Life Science Industry
- Use cases and methods collated by data science professionals from leading companies in the pharmaceutical, agrifood and biotechnology sectors.
- Link: [fairtoolkit.pistoiaalliance.org](https://fairtoolkit.pistoiaalliance.org/)

## FAIR 4 Clin
- A guide about FAIR for Clinical trial and healthcare data
- Link: [fair4clin.org](https://fair4clin.pistoiaalliance.org/)

## Notes
